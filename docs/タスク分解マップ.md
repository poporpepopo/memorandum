## 🛠 会議アシスタント開発：タスク分解マップ

### Phase 1: プロトタイプ（macOS / マイク入力）

#### 1. 環境構築・基盤準備（工数：15%）

* [x] **Ollamaのセットアップ**: `gemma3:4b` モデルのプルと推論確認。
* [x] **Whisperの導入**: `openai-whisper` ライブラリと依存パッケージのインストール。
* [x] **Audioデバイス制御**: `PyAudio` を介したマイクへのアクセス権限確保。

#### 2. 音声キャプチャ・パイプライン（工数：25%）

* [x] **バッファリング設計**: `phrase_time_limit=60` による1分単位の音声切り出し。
* [x] **サンプリング調整**: Whisperに最適な `16,000Hz` へのサンプリング設定。
* [x] **メモリ変換**: バイナリ音声を `NumPy float32` 配列へ変換し、ディスクI/Oを排除。

#### 3. 信号処理・ノイズフィルタ（工数：20%）

* [x] **音圧検知 (VAD)**: NumPyを用いた実効値（RMS）算出ロジックの実装。
* [x] **足切り判定**: RMSしきい値未満を「無音」として後続処理をパスさせる分岐作成。
* [x] **Whisperパラメータ調整**: `no_speech_threshold` 等によるハルシネーション（幻聴）抑制。

#### 4. LLM要約・UI連携（工数：25%）

* [x] **要約プロンプト最適化**: 30〜40文字の制約と、Markdown装飾を除去する命令のチューニング。
* [x] **通知発行**: `plyer.notification` を用いたOSバナーへのテキスト受け渡し。
* [x] **Stdout視覚化**: ターミナル上で原文と要約を対比表示するログ出力処理。

#### 5. 終了処理・データ永続化（工数：15%）

* [x] **シグナルハンドリング**: `KeyboardInterrupt` をキャッチし、正常終了シーケンスへ誘導。
* [x] **全体俯瞰要約**: 蓄積された全原文をLLMに再投入。
* [x] **ファイルI/O**: 日時スタンプ付き `.txt` ファイルへのUTF-8保存。

### Phase 2: アーキテクチャ改善（2026-07 実施）

* [x] **責務分割**: 全処理を `main()` に持つ構成から、`Transcriber` / `Summarizer` / `Notifier` / `MeetingAssistant` へのクラス分割と依存性注入へリファクタリング。
* [x] **並行パイプライン化**: 録音スレッドとAI処理スレッドを `queue.Queue` で分離し、推論中の発言取りこぼしを解消。
* [x] **CLI化**: `argparse` によるオプション体系の整備（モデル・周期・しきい値・保存先など）。
* [x] **フェイルセーフ**: Ollama接続とモデル存在の起動時チェック、通知失敗の隔離、最終要約失敗時の原文保全。
* [x] **型ヒント・docstring整備**: 全公開関数への型注釈と説明の付与。

### Phase 3: Windows / Web会議対応（2026-07 実施）

* [x] **Windows対応**: `requirements.txt` の環境マーカーによる依存切替（pywin32等）、cp932コンソールでのエンコーディング耐性。
* [x] **ループバック録音**: PyAudioWPatch (WASAPI) によるスピーカー出力の直接録音。リサンプリング・モノラル化を実装。
* [x] **音源抽象化**: `AudioSource` Protocol による `SystemAudioCapture` / `MicrophoneCapture` の切替（`--source`、OS別の既定値）。
* [x] **ワンクリック起動**: `run.bat` / `run.command`（初回セットアップ自動実行）、`.gitattributes` による改行コード管理。
* [x] **E2E検証**: 合成音声（SAPI TTS）→ ループバック録音 → Whisper文字起こしの一致確認。

### Phase 4: モデル更新（2026-07 実施）

* [x] **LLM 更新**: 要約モデルを `gemma3:4b` から `gemma4:e4b`（エッジ向け・128K コンテキスト）へ差し替え。低メモリ環境向けに `gemma4:e2b` を案内。
